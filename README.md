# restopia

レストラン情報の総合ポータルサイト

## 概要

restopiaは日本の外食産業に関する様々な情報を提供するWebアプリケーションです。
レストランチェーンの企業グループ関係、満足度ランキング、トレンド分析など、外食に関する総合的な情報を1つのプラットフォームで提供します。

## コーナー一覧

### 1. 外食チェーン相関図
日本の主要な外食チェーン店の企業グループ関係を視覚化したインタラクティブな相関図です。

**特徴:**
- グループごとの色分け表示
- レスポンシブデザイン
- リセット機能
- ラベル表示切替

**含まれている企業グループ:**
- ゼンショーグループ（すき家、なか卯、ココスなど）
- すかいらーくグループ（ガスト、バーミヤン、ジョナサンなど）
- コロワイドグループ（かっぱ寿司、牛角など）
- サイゼリヤ、ワタミグループ、トリドールグループ、吉野家HDなど

### 2. レストラン満足度ランキング
最新の調査データに基づく満足度ランキングを掲載

**データソース:**
- 好きな外食チェーン店ランキング（2024年9月調査、6,204名対象）
- 日本版顧客満足度指数（JCSI）2024年度
- ダイヤモンド顧客満足率調査（2017年）

### 3. その他のセクション（準備中）
- レストランレビュー
- メニュー検索
- トレンド分析
- お得情報
- 店舗マップ


## 使い方

1. `index.html` をブラウザで開きます
2. トップページから各コーナーにアクセスできます
3. 外食チェーン相関図では、企業グループの関係を視覚的に確認できます
4. 満足度ランキングでは、最新のベスト・ワーストランキングを確認できます

## カスタマイズ

### 新しいグループを追加する

`data.js` の `restaurantData.groups` 配列に新しいグループを追加してください：

```javascript
{
    id: 'unique-id',
    name: 'グループ名',
    color: '#カラーコード',
    restaurants: ['チェーン店1', 'チェーン店2', ...]
}
```

### スタイルの変更

`styles.css` を編集して、色やフォント、レイアウトをカスタマイズできます。
